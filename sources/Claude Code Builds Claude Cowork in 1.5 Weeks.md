**Summary**
Claude Code wrote nearly all of the code for Claude Cowork — the first major Anthropic product built primarily by another Anthropic AI tool. Product manager Felix Rieseberg reported the app shipped within one 1.5-week sprint, though prior prototypes existed and Claude Code had already built the extensive backend; the sprint likely covered the frontend.

**Key points**
- Claude Code (Boris Cherny's tool) wrote almost all of the Claude Cowork codebase
- Total ship time: one sprint of ~1.5 weeks from prototype to release
- Prior prototypes and conceptual groundwork existed beforehand
- Claude Code served as the base; sprint focused primarily on the user-facing frontend
- First notable instance of Anthropic shipping a product largely written by its own AI

**Claude's notes**
A clean data point on software development velocity with agentic tools. The 1.5-week sprint is credible given the prior prototype work and the existing Claude Code infrastructure — it's not a claim that a new app was built from scratch. The more interesting signal is that Anthropic felt confident shipping production software primarily written by AI, which normalises the practice internally. Connects directly to Cherny's workflow article (parallel agents, self-verification) and represents the emerging "software writes software" loop.
