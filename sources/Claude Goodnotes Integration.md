**Summary**
Goodnotes integrated with Claude via the Claude Connector directory (claude.ai/directory), allowing users to push Claude conversations directly into editable Goodnotes documents without copy-pasting. Practical use cases: creating a study plan in Claude then sending it to Goodnotes, structuring meeting notes, sharing with others. ChatGPT had already been integrated into Goodnotes previously.

**Key points**
- Claude conversations → Goodnotes documents in a few clicks, no copy-paste
- Integration available in Claude's Connector directory at claude.ai/directory
- Use cases: study plans, meeting note structuring, collaborative documents
- Second major AI integration for Goodnotes after ChatGPT

**Claude's notes**
This is the AI-to-notes pipeline materialising: AI generates structured content, the user exports it to their writing tool. It's a thin integration but it removes meaningful friction. The fact that Goodnotes integrated both Claude and ChatGPT suggests they're treating AI as infrastructure rather than partnering exclusively — the right call for a tool that needs to serve users across AI ecosystems.
