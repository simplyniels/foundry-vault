**Summary**
Google released AI Edge Gallery, an app running large language models entirely on-device (no cloud, no internet required). Available for Android and iOS, it supports chat, image analysis (Ask Image), real-time audio transcription and translation (Audio Scribe), and Agent Skills that connect the model to external tools like Wikipedia or maps. A Thinking Mode visualises intermediate reasoning steps.

**Key points**
- LLMs run locally on device — offline, private, no cloud dependency
- Thinking Mode: model shows intermediate reasoning steps for complex tasks
- Ask Image: camera/gallery analysis processed entirely on device — no uploads
- Audio Scribe: real-time speech transcription and translation on-device
- Agent Skills: extensible tool use (Wikipedia, maps, etc.) from the local model
- Targeted at developers, curious users, and admins experimenting with local LLM capability

**Claude's notes**
On-device AI removes the latency, cost, and privacy concerns of cloud inference — but the model quality gap has historically made it impractical. If Google is shipping this publicly with image and audio capabilities, on-device models have crossed some quality threshold. The privacy angle (images never uploaded) is a strong differentiator for sensitive use cases. Connects to the broader theme of AI moving into every surface: browser, OS, device — not just cloud apps.
