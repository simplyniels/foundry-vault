**Summary**
Todoist published official instructions for connecting its MCP server to ChatGPT, enabling users to read, create, and update tasks and projects via chat. The integration is designed for weekly reviews, project planning, and turning brain-dumps into structured next actions — any clear description translates into Todoist changes.

**Key points**
- Connect Todoist MCP server to ChatGPT via Developer mode; authorise through Todoist
- Supports: read tasks/projects, create tasks/projects, update tasks/projects
- Best use cases: weekly review, project creation with sections, inbox triage, reschedule overdue tasks
- "Anything you can describe clearly, ChatGPT can translate into structured Todoist changes — just skim before confirming"
- Sample prompts: inbox quick wins, brain-dump to next actions, overdue task review, multi-section project creation

**Claude's notes**
A practical example of MCP creating value for end users rather than just developers: natural language becomes the interface to task management. The "skim before confirming" caveat is the right user-experience call — preserving human review on write operations builds trust. This is the productivity-tool expression of the same agentic pattern appearing in Cowork, Dispatch, and Galaxy AI.
