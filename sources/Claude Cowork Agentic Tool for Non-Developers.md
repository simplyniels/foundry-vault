**Summary**
Anthropic launched Claude Cowork, an AI agent aimed at non-programmers for everyday office work. Built on Claude Code but with a consumer-friendly interface, it works by delegating tasks to it like a colleague — sorting files, creating notes, extracting tables from screenshots — while the user retains explicit control over permissions and is notified before larger actions. Initially macOS only; Windows and sync support coming.

**Key points**
- Based on Claude Code but explicitly targets non-developer users
- User grants folder access; Claude works autonomously without constant context re-injection
- Delegation model: tell it what you want done, come back to results — not chat ping-pong
- Permissions are explicit; larger actions are announced before execution (control remains with user)
- Launched as macOS preview; Windows and cross-device sync planned

**Claude's notes**
Cowork represents Anthropic's play for the office worker market beyond developers. The design philosophy — user retains control, agent announces intent before acting — addresses the trust problem that makes most people hesitant to hand autonomy to AI. Worth watching alongside Dispatch (remote control feature) as evidence that Anthropic is building toward a persistent, ambient AI layer for knowledge workers.
