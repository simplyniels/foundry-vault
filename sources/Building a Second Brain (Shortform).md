**Summary**

Shortform's summary of Tiago Forte's Building a Second Brain, condensing the CODE framework and PARA system into highlights. The core claim is that knowledge work — recalling information, making connections, developing novel ideas — is the most valued skill in contemporary society, and a well-organised external knowledge system (ESS) dramatically amplifies this capacity.

**Key points**
- Knowledge work = recalling important info + making connections + developing novel ideas
- Save only information that resonates: inspiring, useful, personally important, or belief-shifting
- Sort information using the PARA hierarchy: current goals → ongoing engagements → areas of interest → archive
- The condensing and organising steps of CODE make information retrievable; distilling and expressing make it usable
- Distill by bolding key points; then bold within bolded — layer-by-layer progressive summarisation
- Intermediate packets are small, named, reusable task bundles (e.g. "research material", "thesis", "paragraph 1")
- Once a goal folder has enough context, outline tasks and sequence subfolders chronologically
