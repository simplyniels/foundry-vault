**Summary**
OpenAI added full Model Context Protocol (MCP) support to ChatGPT, enabling developers to build MCP app interfaces that run directly inside ChatGPT. MCP was originally developed by Anthropic as an open standard for connecting AI models to external tools and data sources; OpenAI's adoption signals cross-industry convergence on the protocol.

**Key points**
- ChatGPT now has full MCP support — Anthropic-originated protocol adopted by OpenAI
- Developers can build MCP app UIs that function natively inside ChatGPT
- Enables deeper integration with third-party tools from within the chat interface

**Claude's notes**
The adoption of an Anthropic-created protocol by its main competitor marks MCP as a genuine industry standard rather than a proprietary Anthropic feature. This is significant: open protocol wins create network effects that benefit all participants. The same dynamic is visible across this ingest batch — Todoist, Codex, Samsung Browser all using MCP for AI-to-tool connectivity.
