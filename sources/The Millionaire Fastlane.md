**Summary**

MJ DeMarco's polemic against the conventional "slowlane" wealth path (education → career → retirement savings) and argument for an entrepreneurial "fastlane" — creating scalable value, building systems that generate passive income, and achieving financial independence while still young.

**Key points**
- There is a hidden path to wealth that operates on speed and leverage rather than incremental savings
- The fastlane is accessible while one is still young and energised, not after decades of accumulation

**Claude's notes**

Single highlight only — the introduction establishing the premise. DeMarco's core distinction (slowlane: work decades, save, retire; fastlane: create scalable value, earn faster) is a useful framework but is represented here by only one sentence. Connects thematically to [[Million Dollar Weekend]] (Kagan's weekend validation model is a fastlane approach) and [[Young Money Guide]] (the latter advocates the slowlane — passive index investing). The two books represent opposite strategic philosophies on wealth-building and would make an interesting conceptual contrast article.
