**Summary**
Anthropic launched Dispatch as a Research Preview inside Claude Cowork. Dispatch enables a persistent AI conversation running on a local machine that can be controlled remotely from a smartphone. The user sends a task from their phone and returns to completed work. All actions require user approval before execution.

**Key points**
- Persistent AI session runs locally on the user's machine
- Controllable from anywhere via smartphone
- Asynchronous pattern: send task remotely, return to completed work
- All actions require explicit user approval before executing
- Launched as Research Preview (early/limited access)

**Claude's notes**
Dispatch is the architectural centrepiece that makes Cowork useful beyond the desk. It changes the value proposition: AI isn't a real-time assistant you must attend to — it's a worker you can brief and check in on. The approval gate is critical for the trust model. Combined with Telegram integration, Dispatch means Claude Code can be part of a workflow that spans mobile → local machine without the user ever opening a terminal.
